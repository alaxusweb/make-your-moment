# 校了チェックリスト

## 内容

- [ ] 対象月の日数と日付シール数が一致
- [ ] 35枠すべて使用
- [ ] 余り枠はテーマ関連のおまけ
- [ ] 35番はテーマと月のコンセプトシール
- [ ] 日付シールだけに`和名 / Romaji`
- [ ] 和名、ローマ字、表題、月に誤字なし

## 題材

- [ ] 立体商標のある近現代建造物を含まない（東京タワー等）
- [ ] 看板・広告に実在の意匠やロゴが写り込んでいない
- [ ] 慰霊碑・災害遺構など倫理的に問題となる題材を含まない
- [ ] 旭日意匠など国際的に係争的な図案を含まない

## 個別素材

- [ ] 35枚すべて独立RGBA PNG
- [ ] 花、葉、小物、白枠に欠けなし
- [ ] 他シール片、孤立線、マゼンタ、透かし、生成文字なし
- [ ] 四隅が透明
- [ ] 暗背景コンタクトシートを原寸確認

## 配置

- [ ] 7列×5行
- [ ] 全ラベルと絵柄が列中心に一致
- [ ] 最低24pxの切断余白
- [ ] 表題は上端の小さな1行
- [ ] 花名がラベル内に収まり、字間が詰まりすぎていない
- [ ] alignment PDFを原寸確認

## PNG

- [ ] 3072×2048px、300dpi、sRGB RGBA
- [ ] 四隅が完全透明
- [ ] PDFをラスタライズした同一レイアウト
- [ ] 単純な低解像度アップスケールではない

## PDF

- [ ] 1ページ、737.28×491.52pt、暗号化なし
- [ ] 日数分の花名だけがテキスト抽出可能
- [ ] 表題とコンセプトは抽出されない
- [ ] 日本語フォントが埋め込み・Unicode対応
- [ ] ラスターデザイン層にalpha soft maskあり
- [ ] PDFを3072×2048pxで描画すると納品PNGと一致
- [ ] alignment PDFだけにガイド線あり
- [ ] PDF編集ソフトで1ラベルを変更・保存・再表示できる

## 販売物

- [ ] `customer/`にPNG、PDF、READMEがある
- [ ] 商品仕様に応じて個別PNGとライセンスを追加
- [ ] `qa/`、manifest、中間物が販売ZIPに混入していない
- [ ] 全面ベクターPDFと誤記していない

自動検査：

```sh
scripts/verify.sh --manifest output/releases/<theme>/<year-month>/manifest.json
```
