# 制作仕様

## 固定値

| 項目 | 値 |
|---|---|
| 用途 | 印刷して切り分けるダウンロード販売データ |
| キャンバス | 3072 × 2048 px |
| 解像度 | 300 dpi |
| 物理寸法 | 約260.10 × 173.40 mm |
| 色・背景 | sRGB RGBA、完全透過 |
| 配置 | 7列 × 5行＝35枠 |
| 最低カット余白 | 24 px（300dpiで約2mm） |
| 必須成果物 | 透過PNG、花名編集用PDF |
| QA成果物 | alignment-check PDF |

1536×1024pxの単純2倍化は不可。個別素材を最終配置サイズ以上で用意し、
3072×2048px基準のレイアウトへ直接組版する。

## 枚数

```text
日付シール = 対象月の日数（28-31）
おまけ総数 = 35 - 日数
ランダムおまけ = おまけ総数 - 1
コンセプトシール = 1（必ず35番）
```

日付シールだけに`和名 / Romaji`ラベルを付ける。おまけにはラベルを付けない。

## 共通生成フロー

1. 個別PNGを7×5へ配置する。
2. 絵柄、表題、白紙ラベル枠、おまけ、コンセプト文字を1枚のラスターデザイン層にする。
3. 日付シールの花名だけをPDF実テキストで重ねる。
4. 1ページPDFを出力する。
5. そのPDFを300dpi相当で3072×2048pxの透過PNGへ変換する。

PDFとPNGを別々に組版してはいけない。

## PDF

- ページ：737.28 × 491.52 pt、1ページ、暗号化なし
- 編集可能：対象月の日数分の`和名 / Romaji`のみ
- ラスタ固定：絵柄、表題、ラベル枠、おまけ、コンセプト
- 花名：Unicode対応の埋め込みシステムフォント
- フォームフィールドではなく通常のPDFテキストオブジェクト
- PDFページには白背景を描かず、ラスターデザイン層のアルファを保持

PDFビューア上でページが白く見える場合があるが、背景を焼き込んだとは限らない。

## 個別素材

- `customer/individual/`に`sticker-01.png`から`sticker-35.png`を置く
- すべてRGBA、透明背景、白いダイカット輪郭付き
- 花や小物を欠かさず、隣画像の断片、マゼンタ、透かし、生成文字を含めない
- 35番はコンセプト文字を置ける余白のある絵柄

## 著作権・販売

- 特定作品のキャラクター、ロゴ、商標、透かしを使わない
- 市販ステッカーや著名作家固有の図案を複製しない
- OSフォント本体を販売ZIPへ同梱しない
- 利用許諾条件は販売先の規約と販売者の方針に合わせ、別途確定する

## 題材選定の3基準

描く前に各題材を次の3点で検査する。疑わしいものは採用せず、差し替えた事実と
理由を記録する。「有名だから」「観光名所だから」は採用理由にならない。

### 1. 商用侵害リスクの排除

ロゴやキャラクターだけでなく、**建造物の形状そのものが立体商標**として登録されて
いる場合がある。

| 不可 | 可 |
|---|---|
| 東京タワー、東京スカイツリー、通天閣 | 城郭、寺社、庭園、古い町並み、橋、自然景観 |
| 企業施設、テーマパーク | 重要文化財、史跡 |

街並みを描く場合、**看板・広告は実在の意匠を写さず創作する**。実在ロゴが1つでも
入ると商用利用できない。

### 2. 倫理面

大量死・惨禍・追悼に結びつく題材は、**明示的な指示がない限り採用しない**。

本商品は手帳を飾る装飾品であり、日付シールは旅行鞄やカメラのおまけと同じシートに
並ぶ。慰霊碑をその文脈に置くと、追悼対象の死を商品化する形になる。原爆ドームが
最も明確な例で、災害遺構・墓所も同じ扱いとする。

**季節性が問題を重くすることがある。** 8月のシートは広島原爆の日を含む。

### 3. 国際的な受け取られ方

本商品は Etsy で全世界に販売される。日本国内では中立でも、国外では政治的主張と
受け取られる図案がある。

- 旭日の放射状意匠（街並みの看板・背景装飾に紛れ込みやすい）
- 領土係争地
- 戦時下の意匠・軍装

日本国内の感覚だけで判断しない。
